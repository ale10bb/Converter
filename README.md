# 自用转换工具集

存放一些转换用的辅助工具，用于摆脱重复劳动。

## 工具集结构

- 核心操作逻辑已打包，放在Converter中；
- 根目录的文件作为功能入口，主要进行参数校验、文件筛选；
- doc的同名文件夹中存放各功能的说明；
